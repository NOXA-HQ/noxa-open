# NOXA Open Resources

This repository contains curated learning resources recommended by NOXA
for building strong foundations in software engineering.

## Purpose
- Help learners avoid tutorial overload
- Share proven, high-quality resources
- Encourage learning by building

## How to use this repo
Pick ONE topic at a time.
Learn → Build → Break → Explain.

## Topics
- Frontend (React, Web fundamentals)
- Backend (Java, Go)
- Computer Science fundamentals
- Developer tools

NOXA believes depth beats speed.
